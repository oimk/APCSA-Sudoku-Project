## Summary of how the Sudoku board is generated
The Sudoku Board is generated by, first, filling in the diagonal 3x3 matrices. These matrices are filled in with random unique numbers from 1-9. After these matrices are finish generating, the program process to recursively generate the remaining 3x3 matrices filled with 0. For each zeroed element in the grid, the program generates a scrambled list of 1-9, as the solutions. The program runs through each element inside that list, and checks if that current element is vaild, according to Sudoku's rules. If the number is not, then the program proceeds to the next number. If the numbers run out, then the program goes back to the previous element, and tries the next number in the previous element's list. If the number does work, then the program places that number and process to the next element in the board until all zeros in the board are filled.
