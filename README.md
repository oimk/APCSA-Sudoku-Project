Project Requirments:
Java Source File
Read me
Design Document

Part 1 Required Features (Mandated for everyone):

Your Java console application must do all of the following:
Create a 9 by 9 solved Sudoku puzzle
Store the puzzle in a 2D array
Use a 2D array and an array and/or ArrayList in a meaningful way
Print the completed solved Sudoku board neatly to the console
Include methods to support your design
Be understandable and well organized

How to run the program
Summary of how the Sudoku board is generated
List of files included in the project
